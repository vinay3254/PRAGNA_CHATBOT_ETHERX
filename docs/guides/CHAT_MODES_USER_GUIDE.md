# Chat Modes - User Guide

## What are Chat Modes?

**Chat Modes** let you choose how the AI assistant should respond to you. Once you select a mode, ALL your messages will be answered in that style until you change to a different mode.

## How to Use

### Step 1: Select a Mode
At the top of the chat interface, you'll see 7 mode buttons:

```
💬 General | 📚 Explain | 💡 Ideas | ✍️ Write | 💻 Code | ❓ Questions | 🎭 Story
```

Click any button to activate that mode.

### Step 2: The Button Highlights
Once selected, the button will turn **golden (highlighted)** to show it's active. This means all your future messages will be answered using that mode's style.

### Step 3: Chat Normally
Type your message and send it. The AI will respond according to the selected mode.

### Step 4: Switch Modes Anytime
- Click a different mode button to switch to that style
- Your chat history stays the same
- All new responses will use the new mode

## The 7 Chat Modes Explained

### 💬 **General Chat**
**Default mode** - Standard helpful assistant

**Best for:**
- Regular questions
- Getting straightforward answers
- General conversation

**Example:**
- You: "What is photosynthesis?"
- Response: Clear, informative answer about how plants convert sunlight

---

### 📚 **Explain Concepts**
**Education mode** - Breaks down complex ideas into simple parts

**Best for:**
- Learning new topics
- Understanding difficult concepts
- Getting step-by-step breakdowns

**Example:**
- You: "What is photosynthesis?"
- Response: 
  1. What is it (definition)
  2. Why plants do it (purpose)
  3. How it works (step-by-step)
  4. Real-world analogy to understand better

---

### 💡 **Generate Ideas**
**Brainstorming mode** - Creative and diverse suggestions

**Best for:**
- Brainstorming projects
- Finding creative solutions
- Getting multiple perspectives
- Innovation and new ideas

**Example:**
- You: "I want to start a blog"
- Response: 10+ creative blog topic ideas with descriptions

---

### ✍️ **Write Content**
**Writer mode** - Professional, polished writing

**Best for:**
- Writing emails
- Creating blog posts
- Crafting articles
- Professional communication

**Example:**
- You: "Write a professional email about a meeting"
- Response: Well-structured, business-appropriate email you can send

---

### 💻 **Code Assistance**
**Developer mode** - Expert programming help

**Best for:**
- Writing code
- Debugging problems
- Learning programming concepts
- Getting code explanations

**Example:**
- You: "How do I sort a list in Python?"
- Response: Clean code examples with explanations and best practices

---

### ❓ **Ask Questions**
**Inquiry mode** - Thoughtful, probing questions to deepen understanding

**Best for:**
- Exploring ideas deeply
- Getting curious perspectives
- Uncovering hidden assumptions
- Deep learning

**Example:**
- You: "Should I change careers?"
- Response: Thoughtful questions to help you explore the decision (rather than just an answer)

---

### 🎭 **Creative Writing**
**Storytelling mode** - Narrative and creative focus

**Best for:**
- Writing fiction stories
- Creative projects
- Imaginative content
- Entertainment writing

**Example:**
- You: "Write a short story about a mysterious door"
- Response: Engaging narrative with vivid descriptions and interesting characters

## Real-World Usage Examples

```
1. Select "Explain Concepts" mode
2. Ask: "How do neural networks work?"
3. Get detailed educational breakdown
4. Keep asking follow-up questions in same mode
5. When you understand, switch to "Code Assistance"
6. Ask: "Show me code to build a neural network"
7. Get programming examples
```

### Example 2: Content Creation

```
6. Get professional article ready to publish
```

### Example 3: Problem Solving
```
1. Select "Ask Questions" mode

3. Get thoughtful questions to help you decide
4. Gain clarity from exploring deeper
```

## Tips for Best Results

✅ **DO:**
- Switch modes to match your current task
- Use "Explain Concepts" for learning
- Use "Code Assistance" when programming
- Use "Write Content" for any writing tasks
- Change modes as your needs change

❌ **DON'T:**
- Forget that your selected mode affects ALL messages
- Use "Code Assistance" if you just want general information
- Expect "Write Content" mode to give you code snippets (use Code Assistance instead)

## Mode Persistence

📍 **Your selected mode is saved locally** - When you come back to the chat, it will remember which mode you last selected.

🔄 **Mode applies to new messages only** - Changing modes doesn't affect previous messages, only future ones.

## Troubleshooting

**Q: My mode isn't highlighted**
- Refresh the page
- Check if you clicked on the button (it should turn golden)

**Q: The mode doesn't seem to be working**
- Make sure you selected a mode before typing
- Try selecting again
- Refresh and try once more

**Q: Can I create custom modes?**
- Currently, only the 7 built-in modes are available
- More custom mode options may be added in the future
